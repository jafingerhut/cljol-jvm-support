# cljol-jvm-support

A Java library that is only a small modification from 4 of the classes
in the [Java Object
Layout](https://openjdk.java.net/projects/code-tools/jol) (JOL)
library, intended to support slightly more accurate Java object
walking than JOL.

## Usage

Build:

```bash
$ lein jar
```
